# Lozan Bags Website

موقع تعريفي وتسويقي متجاوب لمتجر **لوزان للشنط**، يعرض تصنيفات ومنتجات المتجر بطريقة أنيقة وسهلة الاستخدام، مع إمكانية التواصل والطلب مباشرة عبر واتساب.

---

## نبذة عن المشروع

تم تصميم وتطوير الموقع ليكون واجهة رقمية حديثة لمتجر لوزان، ويعرض مجموعة متنوعة من:

- الشنط النسائية
- الشنط الجامعية
- الشنط المدرسية
- المحافظ والإكسسوارات

يركز التصميم على البساطة، وضوح المحتوى، وسهولة تصفح المنتجات من أجهزة الديسكتوب والموبايل.

---

## أهم المميزات

- تصميم عربي كامل باتجاه RTL
- تصميم متجاوب مع مختلف أحجام الشاشات
- عرض تصنيفات المنتجات
- تصفية المنتجات حسب التصنيف
- إضافة المنتجات إلى المفضلة
- روابط مباشرة للتواصل عبر واتساب
- قسم لآراء العملاء
- قسم لعرض حساب المتجر على إنستغرام
- زر للعودة إلى أعلى الصفحة
- قائمة تنقل خاصة بالموبايل
- واجهة هادئة ومتناسقة مع هوية المتجر
